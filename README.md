# 404 Page

A polished custom 404 experience built with Next.js App Router, TypeScript, Tailwind CSS, Framer Motion, and Lucide React.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the custom 404 page.

## Build

```bash
npm run build
```

## Notes

- The homepage intentionally renders the custom 404 experience.
- The main implementation lives in `app/not-found.tsx` and `app/not-found-scene.tsx`.
